# X Thread

Write a thread for X (Twitter) — a series of connected tweets that tell a story or break down an idea.

## Format

- 3-8 tweets, each under 280 characters
- Number each tweet: 1/, 2/, 3/, etc.
- Separate tweets with a blank line

## Structure

- First tweet is the hook — bold claim, surprising fact, or a question that makes people stop scrolling. This tweet does all the work. If it doesn't land, nobody reads the rest.
- Each tweet should stand alone but build a narrative arc when read in sequence.
- End with a takeaway, a question for the reader, or a link to go deeper. Not a CTA begging for retweets.

## Style

- Write like you're explaining something to a smart friend, not an audience
- Short sentences. Punch. Rhythm matters more here than anywhere else.
- Use line breaks within tweets for readability
- Concrete examples over abstract claims

## What to avoid

- Hashtags in the thread body — save 1-2 for the last tweet if relevant
- "Thread" or "A thread" in the first tweet (people can see it's a thread)
- Filler tweets that don't advance the story
- Engagement bait ("Like and RT if you agree")
- Starting every tweet with "And" or "But"
