# YouTube Script

Write a video script that sounds natural when read aloud — conversational, punchy, and visual.

## Format

- 2-4 minutes when read aloud (~300-600 words)
- Include [B-ROLL: description] markers wherever a visual cut or screen recording would enhance the point
- Write for speaking, not reading — short sentences, natural rhythm, contractions

## Structure

- Open with the hook: what the viewer will learn or why this matters. You have 5 seconds before they scroll. Make them count.
- Set up the problem or context in 2-3 sentences. Don't over-explain — get to the meat.
- Walk through the main content. Each point should feel like a natural next step, not a slide in a deck.
- Include [B-ROLL: description] markers for demos, code walkthroughs, diagrams, or UI shots
- End with a clear next step — what to try, what to watch next, where to go deeper. Keep it to one thing.

## Style

- Conversational — "here's the thing", "so what does this actually look like", "let me show you"
- Written for the ear, not the eye. Read it aloud. If it sounds stiff, rewrite it.
- Use "you" and "we" — talk to the viewer, not at them
- Vary sentence length. Short punch. Then a longer explanatory sentence that gives context. Then short again.

## What to avoid

- "Hey guys, welcome back to the channel" openers (get to the content)
- Reading a blog post aloud — video scripts need different rhythm and structure
- Long unbroken monologues without visual markers
- "Don't forget to like and subscribe" in the middle of the content (save it for the end if at all)
- Passive voice and complex sentences that don't survive being spoken
