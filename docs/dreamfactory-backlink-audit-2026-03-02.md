## Gap Analysis

### 1. Stack Overflow — Completely Absent (CRITICAL)

Zero answers linking to DF for high-traffic questions like:
- "auto-generate REST API from database"
- "how to create API for MySQL without coding"
- "REST API generator tool"
- "instant API from PostgreSQL"
- "convert database to REST API"

These questions exist and get thousands of views monthly. Each answer = permanent backlink that compounds.

**Action:** Identify 10-15 relevant SO questions, write genuine helpful answers that reference DF.

### 2. Awesome Lists — Missing From Major Repos

DF is on some awesome-laravel forks but NOT confirmed on:
- `awesome-selfhosted/awesome-selfhosted` (200k+ stars)
- `public-apis/public-apis`
- `awesome-api` lists
- `awesome-rest` lists
- `awesome-database` or `awesome-devops` lists
- `awesome-nodejs` (if DF has Node.js relevance)

**Action:** Submit PRs to each relevant awesome list. Free, permanent, high-DA dofollow links.

### 3. Competitor Comparison Pages — DF Doesn't Own the Narrative

**DF has:**
- Hasura vs DreamFactory (blog.dreamfactory.com)
- Hasura Alternatives (blog.dreamfactory.com)
- Dev.to cross-post of Hasura comparison

**DF is MISSING:**
- DreamFactory vs PostgREST
- DreamFactory vs Supabase (only on SourceForge, DF doesn't control it)
- DreamFactory vs Strapi
- DreamFactory vs Directus
- DreamFactory vs Appwrite
- DreamFactory vs Xano
- DreamFactory vs Retool

**Action:** Create comparison blog posts for each missing competitor. These earn organic backlinks when people search "[X] vs [Y]" and link from forums.

### 4. Developer Publications — Zero Presence

High-DA publications that accept guest posts:
- **The New Stack** (DA 80+) — zero DF content
- **InfoQ** (DA 80+) — zero DF content
- **DZone** (DA 75+) — zero DF content
- **Hacker Noon** — zero DF content

**Action:** Pitch 1-2 guest articles. Topic: "API-first architecture for enterprise data" or "Why auto-generated APIs beat hand-coded ones." Nic's byline + Head of Engineering title = credibility.

### 5. Partner/Integration Directory Pages — Untapped

DF supports 20+ databases. Most vendor directories unclaimed:
- MongoDB partners/integrations page
- Oracle technology partners
- Snowflake partner directory (already on marketplace, but partner page?)
- IBM Db2 integrations
- Cassandra ecosystem
- SQL Server community

**Action:** Apply to each vendor's partner/integration program. Link from their site = DA 90+ backlinks.

### 6. ProductHunt — Absent

No ProductHunt listing found. Even for an established product, a "DreamFactory 6.0" or "DreamFactory for AI" relaunch generates social signals + dofollow links.

**Action:** Plan a PH launch around a major feature release.

### 7. Tutorial/How-To Content Gaps

DF blog has some tutorials but is missing SEO-optimized versions for:
- "How to build a REST API for PostgreSQL" (huge search volume)
- "How to create API for SQL Server"
- "How to connect Oracle database to REST API"
- "Database to API tool comparison 2026"
- "Self-hosted API platform"

**Action:** Create SEO-targeted tutorial posts for each database DF supports. Each post = a page that earns organic backlinks from developers linking in their own blog posts.

## What NOT to Waste Time On

- Generic link exchanges or directories — low value, high noise
- PBN (private blog network) links — Google penalizes these
- Press releases for backlinks — DA is garbage on wire services now
- Social media links — nofollow, don't count for SEO
- Paid link schemes — risky, not sustainable
